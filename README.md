# node-jasmine-demo
快速创建一个node项目
####npm init
>创建一个package.json文件

####git init
>初始化git

####jasmine init
>初始化jasmine

####npm install jasmine --save
>安装依赖，创建一个名为node_modules的文件夹，依赖模块被自动加入到了package.json文件中

####touch .gitignore
>创建.gitignore

####https://www.gitignore.io/docs
>复制系统对应的安装命令，安装gi

####gi linux,node,webstorm >> .gitignore
>添加指定内容到。gitignore文件

####mkdir main
####touch app.js

####cd spec
####touch main-spec.js

>运行测试代码
>npm test
>或 jasmine spec/main-spec.js
